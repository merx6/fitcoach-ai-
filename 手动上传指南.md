# 手动上传文件到 GitHub

由于网络连接问题，我们改用手动上传的方式。

---

## 📋 步骤

### 1. 打开 GitHub 仓库
访问：https://github.com/merx6/my-fitness-app

### 2. 上传文件

1. 点击仓库页面的 **Add file** 按钮
2. 选择 **Upload files**

### 3. 上传这些文件

需要上传的文件都在 `ai-coach` 文件夹中：

**文件路径** | **说明**
---|---
`C:\Users\yezhenzhen\WorkBuddy\20260318141233\ai-coach\index.html` | 主页面
`C:\Users\yezhenzhen\WorkBuddy\20260318141233\ai-coach\app.js` | 应用逻辑
`C:\Users\yezhenzhen\WorkBuddy\20260318141233\ai-coach\style.css` | 样式文件
`C:\Users\yezhenzhen\WorkBuddy\20260318141233\ai-coach\test-script.js` | 测试脚本

### 4. 上传步骤

1. **选择文件**：点击上传区域的 "choose your files" 按钮
2. **多选文件**：一次性选择上面 4 个文件（可以按住 Ctrl 多选）
3. **填写信息**：
   - Commit message 输入：`Initial commit: FitCoach AI 应用`
4. 点击 **Commit changes** 按钮

---

## ✅ 上传完成后

文件上传成功后：

1. 仓库页面会显示这 4 个文件
2. 进入 **Settings** → **Pages**
3. 启用 GitHub Pages：
   - **Source**: 选择 `Deploy from a branch`
   - **Branch**: 选择 `main`
   - **Folder**: 选择 `/(root)`
4. 点击 **Save**
5. 等待 1-3 分钟

---

## 🌐 访问地址

部署完成后，访问：
```
https://merx6.github.io/my-fitness-app/
```
